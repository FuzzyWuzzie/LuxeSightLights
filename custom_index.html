Luxe implementation of Sight & Light by the wonderfully talented Nicky Case

Move the mouse to change the lightsource position. Press R to re-randomize the walls.

Beside the concepts behind producing a 2D "shadow mask" like this, the demo source also shows how to implement custom geometry (in this case, using a triangle fan to cut down on the number of vertices pushed to the GPU [which probably isn't a bottleneck here anyway, but hey.])
